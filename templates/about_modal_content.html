How to Use:

Filter Papers: Use the year range, minimum page count, and the search bar at the top. Press "Apply" (appears on change) for server-side filters.

Checkbox filters: Use checkboxes like "Hide Offtopic", "Hide X-Ray", or "Only Survey" for quick client-side filtering. As the checkboxes have no permanent labels, hover each checkbox to identify the purpose for each.

Classify/Verify: Click the "Classify this paper" or "Verify this paper" button in the detail row to run LLM-based classification on a single paper. Use the top buttons in the "Batch Actions" menu to classify or verifyremaining or all papers.

Edit Data: Click on any âœ”ï¸ âŒ â” symbol in the main table to cycle its value (this is automatically saved on change). Click 'show' in the details column to edit text fields (like "Model Name" or "Other Defects"). These have to be saved manually by clicking on "Save Changes". Unavailable inside HTML exports.

Search: Type on the search bar for instant search on all relevant fields. User comments are searchable. Classifier/Verifier traces are not (to avoid false-positives/noise)

View Statistics: Click the "View Statistics" button to see charts and lists of repeating journals, authors, keywords, and mentioned features/models as well as relevant charts. The stats are calculated based on currently visible filtered data.

Export and Backup: Use the "Export HTML" or "Export XLSX" buttons inside the "Export & Backup" menu to download the currently filtered dataset, or use the backup / restore buttons to work woith full *.parÃ§a.zst backups. Backups include the database, original and annotated PDFs, HTML export and a XLSX spreadsheet. Unavailable inside HTML exports.

Import Data: Use the "Import BibTeX" button to add new papers to the database from a .bib file. Unavailable inside HTML exports.

Attach PDF: Click on the â” in the PDF column of a paper to upload a PDF to it. Unavailable inside HTML exports.

Developed by Luis Alfredo da Silva, to assist his own Master's degree research at Universidade do Estado de Santa Catarina (UDESC).

The following open source libraries, and its dependencies, are leveraged by this software:

Backend: Python, Flask, SQLite3

Frontend: HTML5, CSS3, Vanilla JS

Data Visualization: Chart.js

PDF View/Annotation/Autosave: PDF.js

Utilities: rcssmin, rjsmin (minification), openpyxl (Excel export), pako (compressed HTML export), Zstandard (backup/restore).

Classification and verification was inferenced by Unsloth's Dynamic quants of Qwen3-30B-A3B-Thinking-2507 model from Qwen/Alibaba Cloud.

Local Inference engine: llama.cpp.

Original papers data (BibTeX) was sourced from Scopus, IEEE Xplore (some) and ACM Digital Library
